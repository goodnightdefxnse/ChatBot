Для запуска приложения, необходимо скачать папку "Chat-Bot", где внутри находятся все файлы для необходимой работы приложения, после открыть файл под названием : "chat-bot", ниже инструкция по использованию приложения "Чат-Бот".

Команды для чат-бота :

/start — начало общения с чат ботом. 

/name: (введите ваше имя).

/number: 7, 9 (введите любые числа).

После ввода чисел Чат-бот предложит ввести одно из действий: -, +, *, /

После ввода действия, Чат-бот возвращает результат

/stop — конец общения с чат ботом, если хотите поговорить еще, то введите команду /start

Если была введена команда не указанная в инструкции, Чат-бот ответит: «Я не понимаю, введите другую команду!»
